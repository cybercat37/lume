# Axom VS Code Extension (Local)

This folder contains a minimal VS Code extension that provides syntax
highlighting for `.axom` files.

## Install (local)

1. Open VS Code.
2. Open the Command Palette.
3. Run: `Developer: Install Extension from Location...`
4. Select this folder: `tools/vscode-axom`.

## Notes

- This is a local, unpacked extension (not published).
- Comments are intentionally not supported by the language and are not highlighted.
