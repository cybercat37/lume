# Axom Examples Index

Runnable top-level examples are numbered to keep a stable progression.

Run one from source:

```bash
dotnet run --project src/axom -- run examples/001_demo-run.axom
```

- `examples/demo/` split tutorial snippets.
- `examples/http/` serve + route examples (`examples/http/README.md`).
- `examples/modules/` module/import scenarios (`examples/modules/README.md`).
- `examples/nuget-tool/` tool install/run example (`examples/nuget-tool/README.md`).
